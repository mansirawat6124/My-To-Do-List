# Advanced To-Do List 
A web app where you can:
- Add tasks with a date/time
- Get reminders
- Switch between light & dark mode
- Save tasks even after closing the browser

# How to Use
1. Open `index.html` in a browser
2. Type your task
3. (Optional) Pick a date/time
4. Click Add
